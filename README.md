## Installation (Linux/Mac)
1.Install Python.

2.Clone the repository to your local environment.

3.Move to the project directory in the terminal.

4.pip install required library with command below:
```bash
    pip install -r requirements.txt
```

5.Run program with command below:
```bash
    python main.py  
```

## How to Use
1.Press "start camera" button to open the camera on the pc.

2.Press button of a parts of the corresponding to the facial feature you want to detect.

3.Close camera with "close camera" to stop the camera.
